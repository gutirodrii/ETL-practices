# ECO Practices

Estructura modular por lección/práctica.

## Estructura

- `Lesson-n/Practice-n/`
  - `.venv/`
  - `venv/`
  - `ECO Lesson n Practice n.pdf`
  - `ECO_Lesson_n_Practice_n_solution.ipynb`
  - `requirements.txt`

## Editor Online (Jupyter)

[![Open In Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/gutirodrii/ECO-practices/HEAD?urlpath=lab/tree/Lesson-n/Practice-n/ECO_Lesson_n_Practice_n_solution.ipynb)

Binder instalará las dependencias desde el `requirements.txt` raíz, que referencia el de la práctica.
